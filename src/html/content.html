Features

JS DoX offers 3 main features right now:

  • load additional html elements
  • process a text processor loading elements and inserting calculated values
  • router a simple url hash router

router()

The router uses url hash fields to update the target area. To enable the router just call JSDoX.router(); in a script tag at the end of the page. The router will update the first element with a jsdox-router attribute. A default value/page can be set by setting the attribute to a valid path.
